## 简介


文档采用TexStudio工具编写，xelatex进行编译，我常用的编译环境是Ubuntu 12.04 LTS，Ubuntu 16.04 LTS

，Fedora 24，Mac OS X,其他平台编译调整一些参数即可，编译命令如下：

```latex
 xelatex -synctex=1 -interaction=nonstopmode "main".tex
```
